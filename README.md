## Individuell uppgift - Machine Learning - California Housing
Analysera data från datasetet housing.csv.
Resultatet presenteras i Jupyter Notebooken housing_analysis.ipynb

### Miljö

* Python: 3.13.7

## För att köra projektet
### Klona projetet
```
git clone https://github.com/Cym3ra/ml_assignment1.git
```
## Skapa och aktivera virtuell miljö
```
python -m venv .venv
```
### Windows PowerShell
```
.venv\Scripts\Activate
```
### macOS/Linux
```
# source .venv/bin/activate
```
### För att installera nödvändiga paket:
```
pip install -r requirements.txt
```
